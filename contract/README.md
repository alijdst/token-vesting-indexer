# Token Vesting Smart Contract

Minimal production-style token vesting contract built with Solidity + Foundry.

## Features

- Linear token vesting
- Owner-controlled allocations
- Partial claiming
- Event emission
- Foundry test suite
- OpenZeppelin integration

## Stack

- Solidity
- Foundry
- OpenZeppelin

## Install

```bash
forge install
```

## Run Tests

```bash
forge test -vvv
```
